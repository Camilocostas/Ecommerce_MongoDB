## 📋 Descripción

Taller práctico de consultas con MongoDB Aggregation Framework, aplicado a un dominio de e-commerce (productos, órdenes, clientes). Incluye ejemplos de pipelines usando etapas como `$match`, `$group`, `$lookup`, `$project`, `$sort` y `$limit`.

## 🗂️ Estructura

```
Taller consultas PIPELINE/
└── (consultas de agregación en MongoDB)
```

## ▶️ Cómo ejecutar las consultas

**Opción 1 — MongoDB Shell (mongosh)**
```bash
load("Taller consultas PIPELINE/Consulta 1.js")
```

**Opción 2 — MongoDB Compass o Atlas**
Copia y pega el contenido del pipeline directamente en el editor de agregaciones.

**Opción 3 — Node.js**
```bash
npm install mongoose
node <archivo-de-consulta>.js
```

Requiere una instancia local de MongoDB o conexión a MongoDB Atlas.
